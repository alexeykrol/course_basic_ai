# course-ai_basic

Курс по практическому использованию ИИ в формате Markdown.

## Состояние репозитория

- Уроки очищены от служебного WordPress-контента и технического мусора экспорта.
- Внутренние оглавления переведены на локальные якорные ссылки.
- Для основных уроков добавлена навигация между файлами (предыдущий/следующий).

## Как читать

1. Идите по порядку из раздела `Карта уроков`.
2. Внутри каждого урока используйте `Оглавление` и блок `Навигация по курсу`.
3. Для ускоренного прохождения начните с урока 17 (сжатый практический конспект), затем углубляйтесь по профильным урокам.
